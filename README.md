📚 مشروع Book Buddy - نظام إدارة المكتبة الذكي
قسم علوم المعلومات - جامعة بني سويف
مشرع تخرج متميز يهدف إلى تطوير بيئة المكتبات الأكاديمية باستخدام أدوات الذكاء الاصطناعي وتحليل البيانات لدعم اتخاذ القرارات المعلوماتية.

👤 فريق العمل
قائد الفريق: إسلام ربيع عبد الحميد
تحت إشراف: د. وسام الوكيل & د. اميرة محمد
🛠 التكنولوجيات المستخدمة
Frontend: React.js / Vite / Tailwind CSS
Backend: Supabase (PostgreSQL)
AI Integration: Groq API
Deployment: Vercel
🚀 المميزات الرئيسية
تحليل البيانات: لوحة تحكم ذكية لعرض إحصائيات التفاعل والكتب.
المساعد الذكي: شات بوت مدعوم بالذكاء الاصطناعي لمساعدة الباحثين.
إدارة الخدمات: تنظيم الإعارة، الوصول الرقمي، والرسائل العلمية.
